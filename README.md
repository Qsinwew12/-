《戏字小助手》是一款专为一至六年级设计的华文写字互动教学工具，集同步学习、互动练习与智能评分为一体，让写字教学更有趣高效。  核心亮点：  课本精准同步：按年级、单元和课文筛选专属生字。全方位解析生字（拼音、部首、笔画、释义、配词与语音）。  多元互动练习：提供“严格笔顺”与“描红临摹”模式。教师可进行大屏楷体示范与手动圈画点评，指导更直观。  智能评分反馈：系统从字形大小、重心、间架结构等维度自动评分并精准标记问题。提供分层评语（低年级易懂，高年级详尽），评分结果直接对标国家课标的“表现标准 1 至 6”。  版权声明：本工具为丘欣怡与吴巧滢原创作品（Hak Cipta Terpelihara © 1987），仅限学习与教育用途，严禁复制、修改或作商业用途。
